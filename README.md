A highly responsive **Admin Dashboard** interface built primarily with **Bootstrap**, utilizing minimal custom **CSS3** and semantic **HTML5**. This project, titled "Ghost Admin," focuses on delivering a clean, mobile-first user experience for data-driven backend systems.

---

## ğŸŒŸ Key Features

* **Bootstrap-Centric Design:** Built almost entirely with Bootstrap components and utility classes for a consistent and professional UI.
* **Fully Responsive:** Leverages the Bootstrap Grid system to ensure the dashboard is fully functional and visually appealing on mobile, tablet, and desktop screens.
* **Quick Stats Overview:** Features four specialized cards for tracking Daily, Weekly, Monthly, and Yearly visitor metrics.
* **Dynamic Sidebar Navigation:** A clean, vertical navigation menu organized by "Controls" (Dashboard, Users, Statistics) and "Actions" (New User, Update Data).
* **Data Presentation:**
    * **Progress Bars:** Visualizes location-based data metrics (Regional, Global, Local, Internal).
    * **Data Table:** A structured table for monitoring age groups and user progress status.
* **Alert System:** Includes a Bootstrap-styled alert banner for system notifications and updates.
